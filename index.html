Plataforma de Dispositivos IOT

O projeto correpsonde a uma plataforma IoT completa onde clientes, com níveis diferentes de acesso ao sistema, podem realizar autenticação, criar empreendimentos com vários ambientes e configurar diversos devices diferentes para estes ambientes. Atualmente existem 5 categorias diferentes de devices que permitem formas diferentes de visualizar dados.

Por exemplo, o DEMETER-WiFi permite visualizar a geolocalização do dispositivo num mapa interativo. Já que foi pensado a um device IOT que monitora gado no pasto. Já o Hestia, controlador IoT de irrigação, permite acionar relés e sistemas de irrigação automáticos. O projeto faz uso de:

  • Python PAHO-MQTT (MQTT Client)
  • Python Flask (Controle de Rotas e autenticação de usuários)
  • Banco de dados MySQL
  • React (Elaboração do front-end)
  • React-bootstrap (Elaboração de Layouts)

NLW E-sports

O projeto foi desenvolvido ao longo da imersão NLW (Next Level Week) promovido pela instituição de ensino RocketSeat ao longo do dia 11/09/2022 ao dia 18/09/2022 que visou elaborar um projeto capaz de envolver conceitos de front-end e back-end com o typescript. Ao longo da trilha, atividades práticas foram realizadas envolvendo:

  • Construção de um banco de dados relacional com o Prisma com duas tabelas (Ad e Games)
  • Construção de um api restful com typescrip
  • Construção de uma landing page com React
  • Construção de um mobile app com React Native
